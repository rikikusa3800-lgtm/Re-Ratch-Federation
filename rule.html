Re:Ratch Federation憲章

※ここではRe:Ratch Federationを仮称として連盟と呼称する。

※前提としてこの憲章に沿っていればチームはどのようなルールも作成でき、メンバーがどのようなことをしても構わないものとします

❏ 基本理念
  • 1.当連盟は以下の組織構造によってスクラッチの活性化などを行い全盛期補填を目的としている。
  • 2.原則として連盟外の荒らしなどの対策は行わない。
  • 3.活動は常に楽しむことを第一とする。
  • 4.連盟はコミュニティーガイドラインを尊重して行動する。
❏ 組織構造

・チームと管理局に分かれて行われる。チームはチームメンバーとチームリーダーとわけ、全盛期補填のプロジェクトに直接関与する。管理局は連盟の原則などを変えたり、チームに助言を与えたり、外部との外交を主任とする。

◎定義
  • 連盟メンバー:本部サイトに記載されているメンバー
  • チーム:マニュアルに沿って作られ、本部サイトに記載されているもの
  • 連盟メンバー:マニュアルに沿って連盟に参加し、本部サイトに記載されているメンバー
  • チームリーダー、管理局: 本部サイトに記載されているメンバーがその対象である。
  • 決議: 議論の最終的な決定を行う方法
  • 独裁行為: 憲章の他メンバーの権利を無視する。決議を行わずに個人的、感情的に物事の決定を行うこと
  • 自治行為: 管理局法務部、運営部以外のメンバーが取り締まりや処罰を勝手に行うこと
◎全メンバーの義務・権利

⚫︎義務

  • 社会通念上認められない行為や法律に抵触する行為の禁止
  • 後述の違反行為をしてはならない
  • 特定の政治的・宗教的な主張を連盟内に持ち込んで対立を煽る行為や、それに基づく過激な攻撃行動をしてはならない
  • マニュアルの通りに行動しなければならない
  • 義務行為の厳守
  • 相手の立場や意見、権利を尊重しなければならない
  • 管理局所属ではないのならばチームには原則1つしか所属できない

⚫︎権利

  • 事前に管理局運営部に連絡し、数ヶ月間の活動休止（非アクティブ扱い）することができるが、処分の免除は行われない
  • 意見や提案の自由を認める
  • マニュアルに沿って新たにチームを結成することができる
  • どのチームにも所属しないことを認める
◎通常メンバー
◯チームメンバー

⚫︎義務

  • チームメンバーは管理局、チームリーダーの決議結果を反映しなければならない
  • チームのスタジオに所属しなければならない

⚫︎権利

  • チームメンバーはマニュアルに沿って管理局になることを希望することができる
  • チームメンバーはマニュアルの改正を提案することや管理局の決定に異議を申し立てることで、管理局と議論することができる
  • チームメンバーは、チームに参加申請したスクラッチャーの加入を、チーム全体の決議によって拒否することができる
  • チームメンバーはチームリーダー選挙(マニュアルを参照)を決議で投票できる
  • チームメンバーはチームリーダーに立候補できる
  • チームリーダーを交代することの決議を求めることができる
  • チームリーダーに決議を申し立てることができる
  • チームメンバーは管理局およびチームリーダーに伝えることで辞めることができる
◯チームリーダー

⚫︎義務

  • チームリーダーは憲章を理解できなければならない
  • チームを作成したものが初代チームリーダーにならなければならない
  • チームリーダーは全メンバーの意見を公平に聞き、中立的な立場で議論をまとめなければならない
  • チームリーダーは全ての活動を指揮し、物事を進ませ続けなければならない
  • チームリーダーは基本的に1週間に3日以上の活動を推奨される
  • 会議の決定はチームメンバーの全員が確認できる場所で決議を行わなければならない
  • チームで行動する際には結果報告所にて報告しなければならない
  • 2名以上のメンバーにより決議を求められた際にはそれを行わなければならない
  • スタジオのリーダーになる必要がある
  • チームメンバーがチームリーダー選挙を求めた際に、その選挙を実施するかどうかを決議にかけなければならない

⚫︎権利

  • チームリーダーはマニュアルの通りに決議を設定することができる
  • 物事を決めるために会議をすることができる
  • チームリーダーは決議を行わずに全チームメンバーに伝えることでリーダーを辞めることができる(そのためチームメンバーは次のチームリーダーを考えておくのもいいと思われる)
  • マニュアルに沿って管理局になることを希望することができる
  • チームリーダーはマニュアルの改正を提案することや管理局の決定に異議を申し立てることで、管理局と議論することができる
◎管理局(連盟管理局)
◯管理局全般

⚫︎義務

  • 管理局は憲章を理解できなければならない
  • 管理局は基本的に1週間に3日以上の活動を推奨される
  • チームの行動が義務を違反する以外はそれを止めるよう示したりすることはできない
  • メンバーが議論を求めた場合、それに応じなければならない
  • 処分や削除を行う場合、具体的な根拠を持った説明を必要とする
  • チームが動き続けられるよう、孤立させず一緒に解決するような姿勢をとる
  • 管理局（運営・外交・法務・広報）のメンバーを決める際は、以下の3つのパターン（新規選出、欠員補充、交代）に対して行われ、マニュアルの通りに行わねばならないものとする。又希望者は当憲章を理解でき、自警行為や荒らし行為を行なっていない者のみとする

⚫︎権利

  • チームの行動が違反である場合、それを伝え、止めることができる
◯管理局運営部

⚫︎理念

連盟において人が残るような環境整備、できる限り全員が楽しく運営を続けられるように努める

⚫︎義務

  • 原則として3名の人数制限とする
  • 管理局含む全ての連盟全体の政策を行い、それに伴う責任を負わなければならない
  • 当憲章やマニュアルの根幹に関わる重大な政策は全メンバー決議で決定し、日常的な活動方針は管理局とチームリーダーとの決議で決定する
  • 議論において全員が納得し、合意できるように努力する
  • 条件をクリアした新規参加メンバーをマニュアル通りに本部サイトに所属させなければならない
  • 処分は法務部での決定を持って行わなければならない。依頼された場合、実行を行わなければならない

⚫︎権利

  • 決議により可決された場合にのみ連盟の原則やマニュアルの最終的な更新を行える
  • 管理局全般に司令を送ることができる
  • 本部サイト、本部スタジオの管理権限を有する
  • 3ヶ月間、活動報告（結果報告所への記載）がないチームは、管理局運営部の判断によって「休眠状態」とし、本部サイトから非表示、または解散・統合させることができる。
◯管理局外交部

⚫︎理念

連盟外の他スクラッチャーとで共同で活動し、全盛期補填の実装のためチームに支援を提案する。

⚫︎義務

  • 原則として3名の人数制限とする
  • チームに他スクラッチャーとの交渉を求められた際にはそれに従わなければならない
  • 外交の際、他スクラッチャーの意見を重要視して行う必要がある

⚫︎権利

  • 他スクラッチャーとの外交で連盟管理局であると明言できる
  • 連盟を代表して、外部との外交や交渉を行うことができる
◯管理局法務部

⚫︎理念

連盟が楽しく過ごせる場所であるよう、憲章やコミュニティガイドラインに違反することに対応し、客観的で公平な判断をする。

⚫︎義務

  • 執行の際は当憲章に沿ったものでなくてはならない
  • チームメンバーかチームリーダーによる違反の場合、その属しているチームと共に処分を決めるものとする
  • 権利の範囲は連盟内部であり理念.2に沿って荒らし行為に対応してはならない
  • 違反したメンバーの権利も尊重しなければならない

⚫︎権利

  • 法務部は違反を確認した場合、運営部および該当チームとの協議（または決議）を経て、チームの削除やメンバーの脱退を運営部に執行を依頼することができる
  • 処分や方針を巡る議論において、20日以上合意に至らない場合、または決議が何度も否決されて運営に支障が出る場合、最終的な判断を行うことができる
◯管理局広報部

⚫︎理念

連盟に何が起こっているのか連盟メンバーに伝え、共有して団結力を増やす。また参加希望のメンバーへの初めの看板としての役割を行う

⚫︎義務

  • マニュアルに沿って定期的に新聞を作る必要がある
  • 新聞は客観的な視点で書かなければならない
  • チームの成果を連盟全員に発表する

⚫︎権利

  • 自由な内容で作成することができる
◎チームについて

・チームは連盟の憲章に則っている限り組織として区分する、部門を作ることや役職を作ることなど自由に作り変えることができるが(無論、1名のみも可能とすることができる)、そのチームの形や方針をマニュアルの通りに記載しなければならない

・リーダーが急に辞任・失踪した場合、チームのルールにその対応について記載されていなければ新リーダーが決まるまでは最年長（または運営部）が「暫定リーダー（代理）」を務める。

❏ 各種他ルール

◯もし以下の違反行為をした者は管理局法務部とその者が所属していたチームのメンバーと共に相談して対応を下すものとする。対応議論には明確な客観的証拠かつ対象となった本人への事実確認と、言い分（弁明）を聞く機会の提供を必須事項とする

主な違反行為は以下の通りである

  • 義務・マニュアルの無視（ただし軽微なものについては個人的な注意にとどめる）
  • 他のメンバーに対する暴言、冷笑
  • 独裁行為
  • 自治行為
  • チームリーダー選挙における投票数を意図的に増やしたりする行為
  • 破壊行為や荒らし行為、反応を見るためにわざと悪質な行動をすること
  • 相手のプライバシーを損害する行為
  • 個人攻撃
  • 連盟の活動やプロジェクトの進行を意図的に妨害する行為、または多くのメンバーが不快に感じるスパム行為

以上のことを行った場合以下の処分が検討される

厳重注意(厳密には処分ではない)・警告・チーム強制脱退・管理局、チームリーダー強制脱退・連盟強制脱退

◯ 決議について

決議については以下のテンプレートを使用し、その期限をすぎて結果を記載するまで決議が行われたとされない。

決議{ ...の決議行いたいと思います。今回の議題は...することを決議します。会議の期間は今後2日間(チームなら“又は”連盟全体なら“かつ”とする)5名以上の意見を収集することにて終了し、投票の際には賛成か反対かその異論や質問について任意で理由も含めて返信してください。 }

結果報告{ 決議にご参加いただきありがとうございました。結果として参加者#名、賛成#名、反対#名、その他#名となりました為(可決、否決)となります。 }

※賛成が全体の50%超でのみ可決となるものとする

◯ 連盟のロゴについて

連盟のロゴは2種類あり、中に保存されているものを採用する。色合いを変えることは原則として自由である。名称として紫の円は"Renmei-2025"で名前が記載されたロゴは"Untitled-New-Federation"である。

"Untitled-New-Federation"は連盟全体のロゴとなり、"Renmei-2025"は管理局全域を指すロゴとする
